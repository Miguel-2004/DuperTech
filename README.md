## Instalacion de paquetes

```bash
$ npm install 
```
Si no se logran instalar los paquetes hacerlo de forma independiente.

body-parser
ejs
express
mysql2
nodemon

Con el siguiente comando
```bash
$ npm install "paquete"
```

## Comando para correr la aplicacion web
```bash
$ npm start
```
